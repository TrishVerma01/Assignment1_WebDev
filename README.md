# Calm Space – Web Development Project

This project is a basic multi-page website created using HTML and CSS. It includes a welcome page, a dashboard page, and a music page. The dashboard is designed to show different sections such as mood tracking, a journal, a habit tracker, weekly mood bars, a break section, and emergency contacts.

# Project Pages

1.welcome.html – Introduction page
2.dashboard.html – Main dashboard layout
3.music.html – A page that contains relaxing music links

# Features

## Mood Summary

A circular progress indicator created using a CSS conic-gradient. It shows the overall percentage value for the user's mood.

## Daily Mood Selector

Radio buttons are styled as emoji options. Selecting one highlights it with background color. This is done only with CSS.

## Weekly Mood Overview

Horizontal bars for each day of the week. The width of each bar represents the mood score for that day. It is built with simple divs and width percentages.

## Journal Section

A text area where the user can type daily thoughts or notes. It also includes a submit button.

## Habit Tracker

A table with checkboxes representing different daily habits. The user can check them manually.

## Break Section

A set of images stored locally in an images folder. Each image appears inside a small box along with a short text line.

## Emergency Contacts

A small box with links for phone and email for immediate contact.

## Navigation Bar

A simple navigation menu at the top of each page. It links all three pages together.

# Technologies Used

1.HTML5

2.CSS3 (Grid layout, Flexbox, gradients, box-shadows, text formatting)

No JavaScript is used in this project.

# Purpose of the Project

The purpose of this project was to practice the following:

Structuring a multi-page website

Using CSS Grid for arranging boxes

Using Flexbox for alignment

Designing a dashboard layout

Embedding images and external links

Working with gradients and box shadows

Creating a clean and simple UI without JavaScript

# How to Run the Website

Download or clone the project folder.

Open welcome.html in any web browser.

Use the navigation bar to move between pages.
