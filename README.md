# Calm Space – Web Development Project

This project is a beginner-level multi-page website created using **HTML, CSS, basic JavaScript, and simple PHP**. It includes a welcome/login page, a dashboard page, and a music page. The dashboard contains mood tracking, a journal section, a habit tracker, weekly bars, a break section, and emergency contacts. Journal entries are stored using PHP file handling.

# Project Pages

1. index.html – Welcome + Login page
2. dashboard.php – Main dashboard layout
3. music.html – Music and relaxation page
4. backend.php – Handles login validation
5. save_journal.php – Saves user journal entries
6. journals/ – Folder where user journal text files are stored

# Features

## Mood Summary

A circular progress indicator created using a CSS conic-gradient. JavaScript updates the percentage.

## Daily Mood Selector

Emoji-based radio buttons styled with CSS. Selecting one highlights it.

## Weekly Mood Overview

Horizontal bars built using div widths to represent each day’s mood level.

## Journal Section

A textarea for daily journal entries. When submitted, PHP saves the journal text in a file named after the user’s email.

## Habit Tracker

A simple table with checkboxes for tracking everyday habits.

## Break Section

Image boxes inside an images folder along with captions.

## Emergency Contacts

A small section containing phone and mail links.

## Login Validation

Basic JavaScript checks email input for “@”
PHP validates email and password again before allowing login.

# Technologies Used

1. HTML5
2. CSS3 (Grid, Flexbox, gradients, box-shadows)
3. JavaScript (basic input validation)
4. PHP (simple form handling + file storage)

# Purpose of the Project

The purpose of this project was to practice:

Structuring a multi-page website
Designing layouts with CSS Grid and Flexbox
Creating a dashboard UI
Using basic JavaScript for validation
Using PHP to handle form submissions
Saving data using simple file-writing
Building a clean interface without advanced frameworks

# How to Run the Website

1. Download or clone the project folder.
2. Place the folder inside a local server (Herd, XAMPP, WAMP, etc.).
3. Open index.html or the project URL in any browser.
4. Login using a valid email and password.
5. Use the navigation bar to move between pages.
